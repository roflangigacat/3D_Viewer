1. Реализация

Программа разработана для визуализации каркасной модели в трехмерном пространстве:

Программа разработана на языке C с использованием библиотеки Qt

Сборка и установка программы происходит при помощи Makefile целью Install на рабочий стол пользователя.

В один момент времени может быть только одна модель на экране.

Программа предоставляет возможность:

  • Загружать каркасную модель из файла формата obj (поддержка только списка вершин и поверхностей)
  • Перемещать модель на заданное расстояние относительно осей X, Y, Z
  • Поворачивать модель на заданный угол относительно своих осей X, Y, Z
  • Масштабировать модель на заданное значение

Графический пользовательский интерфейс содержит:

  • Кнопку для выбора файла с моделью и поле для вывода его названия
  • Зону визуализации каркасной модели
  • Кнопки и поля ввода для перемещения, поворота и масштабирования модели.
  • Информацию о загруженной модели - название файла, кол-во вершин и ребер
  • Кнопку для выбора файла с моделью и поле для вывода его названия
  • Кнопку, позволяющую настраивать тип проекции (параллельная и центральная)
  • Кнопки для настройки типа (сплошная, пунктирная), цвета и толщины ребер, способа отображения (отсутствует, круг, квадрат), цвета и размера вершин
  • Кнопку для изменения цвета фона
  • Кнопки для сохранение скриншотов модели и записи GIF-анимации (640x480, 10fps, 5s)

Есть возможность сохранения всех настроек после перезапуска, для этого необходимо поставить "галочку" на Сохранение настроек

2. Графический интерфейс

Все активные зоны обозначены цифрами (1-5)

Первая зона - Информация об объекте - окно, где выводится подробная информация об загруженной модели: показывается имя файла, количество вершин(точек) и ребер.

Вторая зона - Загрузка файла - открывает проводник и позволяет загрузить файлы из модели только с расширением .obj. Помимо этого реализован drag & drop, позволяющий просто перетянуть любой файл в программу, если будет выбран не корректный формат файла, появится ошибка.

Третья зона - Настройки - позволяет выбирать цвет ребер, фона, размер точек, тип проекции и все остальное описаное выше.

Четвертая зона - Сохранение - если поставить "галочку", то после перезапуска все выбранные настройки будут применены повторно.

Пятая зона - GIF & Screenshot - описание во вкладке Дополнительные настройки.
